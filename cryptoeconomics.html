Enabling Architecture
An architecture for transforming existing systems into enterprise blockchain systems using a phased approach

Read more...

Features

  • 1. Sign-On

    Add Fabric identities to existing user credentials

  • 2. Permissions

    Replace direct access to database with blockchain access and logging

  • 3. Cryptographic Link

    All data to be hashed and copies of the hash written to db and blockchain

  • 4. DBA Auditing

    Write the DBA operations to Fabric audit log

  • 5. Replace legacy datastore

    Move database and logic across to Fabric
